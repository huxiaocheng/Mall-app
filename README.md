# Mall-app
购物商城手机web电商
## npm i 安装相关依赖
## npm run dev 运行
### 脚手架相关命令即可
技术栈采用：vue全家桶+better-scroll滚动库，手写scss，没有使用UI框架，数据为真实接口数据，所有功能全部为手写。
